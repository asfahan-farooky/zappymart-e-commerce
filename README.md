# 🛒 ZappyMart – E-commerce Web Application

ZappyMart is a modern and responsive e-commerce web application designed to provide users with a smooth and engaging online shopping experience. It allows customers to browse products, view details, add items to cart, and place orders easily through WhatsApp integration.

## 🚀 Features

* 🏠 Clean and attractive homepage with hero section
* 🔍 Product search functionality
* 🛍️ Add to cart system with dynamic cart update
* 💳 Checkout system with user details form
* 📱 WhatsApp order integration
* ⭐ Product ratings and reviews UI
* 🖼️ Image gallery with lightbox preview
* 📦 Multiple product categories (clothing, electronics, etc.)
* 📱 Fully responsive design (mobile + desktop friendly)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Font Awesome Icons

1. Download or clone this repository:

```
git clone https://github.com/your-username/zappymart-e-commerce.git
```

2. Open the project folder

3. Run the `index.html` file in your browser

## 📦 How It Works

* Users can browse products and view details
* Products can be added to the shopping cart
* Cart automatically updates total price
* On checkout, user fills details (name, address, phone, etc.)
* Order is sent via WhatsApp for confirmation
